プライバシーポリシー

pbl5班（以下「運営者」といいます。）は、大阪大学の学生間における参考書等の取引支援サービス「OU Textbook」（以下「本サービス」といいます。）における利用者の情報について、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます。）を定めます。

第1条（基本方針）

運営者は、利用者の個人情報およびプライバシーを尊重し、個人情報の保護に関する法律その他の関係法令を遵守するとともに、本ポリシーに従って利用者の情報を適切に取り扱います。

第2条（取得する情報）

運営者は、本サービスの提供にあたり、次の情報を取得することがあります。

1．利用者が登録または入力する情報

  1. 氏名またはニックネーム
  2. メールアドレス
  3. 大阪大学の学生であることを確認するために必要な情報
  4. 所属学部、学科、研究科、学年その他の所属情報
  5. アカウントのログインに必要な認証情報
  6. プロフィール画像および自己紹介
  7. その他、利用者が本サービス上で入力または登録する情報

2．出品および取引に関する情報

  1. 出品した参考書等の商品名、画像、価格、状態および説明
  2. 購入希望、取引状況および取引履歴
  3. 出品者と購入者との間で本サービス上に送信されたメッセージ
  4. 商品の受け渡し方法、日時、場所その他受け渡しに関する情報
  5. 評価、報告、通報その他利用者が投稿した情報
  6. その他、本サービスを通じた取引に関する情報

運営者は、原則として、商品代金の支払情報、銀行口座情報、クレジットカード情報または電子決済サービスの認証情報を取得、保管または管理しません。ただし、利用者が問い合わせやメッセージなどにこれらの情報を自ら記載した場合、運営者が一時的に当該情報を取り扱うことがあります。

3．問い合わせに関する情報

  1. 問い合わせを行った利用者の氏名またはニックネーム
  2. メールアドレスその他の連絡先
  3. 問い合わせ、相談または通報の内容
  4. 本人確認のために必要な情報
  5. 問い合わせへの対応履歴

4．本サービスの利用に伴って取得される情報

  1. IPアドレス
  2. 端末の種類および端末識別に関する情報
  3. OSおよびブラウザの種類
  4. 本サービスへのアクセス日時
  5. 閲覧したページおよび操作履歴
  6. Cookieその他の類似技術によって取得される情報
  7. エラー情報および不正アクセスに関する情報

第3条（個人情報の利用目的）

運営者は、取得した情報を、次の目的で利用します。

  1. 本サービスを提供、維持および運営するため
  2. 利用者の登録、ログインおよび本人確認を行うため
  3. 利用者が大阪大学の学生であることを確認するため
  4. 出品情報の掲載および検索機能を提供するため
  5. 出品者と購入者との連絡機能を提供するため
  6. 取引状況を表示および管理するため
  7. 利用者からの問い合わせ、相談および通報に対応するため
  8. 利用規約に違反する行為、不正利用、詐欺その他の問題を調査および防止するため
  9. 本人確認、アカウントの利用停止、投稿の削除その他必要な対応を行うため
  10. システム障害、不具合およびセキュリティ上の問題に対応するため
  11. 本サービスの利用状況を分析し、機能および利便性を改善するため
  12. 本サービスに関する重要なお知らせ、規約変更および運営上必要な連絡を行うため
  13. 利用者間のトラブルについて事実確認を行うため
  14. 法令または大学の規則に基づく対応を行うため
  15. 前各号の利用目的に付随する目的のため

第4条（利用者間における情報の表示）

  1. 運営者は、参考書等の取引に必要な範囲で、利用者のニックネーム、所属情報、プロフィール、出品情報、評価その他運営者が必要と判断する情報を、他の利用者に表示することがあります。
  2. 氏名、メールアドレスその他の連絡先については、取引または本人確認のために必要な場合を除き、原則として他の利用者に公開しません。
  3. 利用者は、本サービス上で公開した情報が他の利用者に閲覧される可能性があることを理解した上で、投稿する情報を自ら判断するものとします。
  4. 利用者は、住所、電話番号、学籍番号、銀行口座情報、パスワードその他公開する必要のない情報を、出品ページや公開プロフィールに記載しないものとします。

第5条（個人データの第三者提供）

運営者は、次のいずれかに該当する場合を除き、本人の同意なく個人データを第三者に提供しません。

  1. 法令に基づく場合
  2. 人の生命、身体または財産を保護するために必要であり、本人の同意を得ることが困難である場合
  3. 公衆衛生の向上または児童の健全な育成のために特に必要であり、本人の同意を得ることが困難である場合
  4. 国の機関、地方公共団体またはその委託を受けた者が、法令に定められた事務を行うことに協力する必要があり、本人の同意を得ることで当該事務の遂行に支障を及ぼすおそれがある場合
  5. 不正行為、詐欺、脅迫、盗難その他の犯罪のおそれがあり、警察、裁判所その他の公的機関から適法な照会を受けた場合
  6. 本人からあらかじめ同意を得た場合
  7. その他、法令によって認められる場合

なお、利用者間の取引に必要な範囲で、一方の利用者の情報を取引相手に表示または提供することについては、第3条に定める利用目的の範囲内で行います。

第6条（個人情報の取扱いの委託）

運営者は、本サービスの運営に必要な範囲で、サーバー管理、メール送信、アクセス解析、本人確認その他の業務を外部の事業者に委託し、個人情報の取扱いを委託することがあります。この場合、運営者は、委託先を適切に選定し、必要な契約を締結するなど、委託先における個人情報の取扱いを適切に監督します。

第7条（外部サービスの利用）

  1. 本サービスは、ログイン、メール送信、データ保管、アプリケーションの公開その他の目的で、外部サービスを利用します。
  2. 外部サービスの利用に伴い、当該サービスの提供者が利用者の情報を取得する場合があります。
  3. 外部サービスにおける情報の取扱いについては、各外部サービス提供者の利用規約およびプライバシーポリシーが適用されます。

Supabase（Supabase, Inc.）
利用目的：利用者認証、本人確認メールの送信、データおよび商品画像の保存

Render（Render Services, Inc.）
利用目的：本サービスのホスティングおよび配信

第8条（Cookie等の利用）

  1. 本サービスは、利用者のログイン状態の維持、本サービスの利用状況の把握、利便性の向上および不正利用の防止のため、Cookieその他の類似技術を使用することがあります。
  2. 利用者は、ブラウザの設定によりCookieを無効にできます。
  3. Cookieを無効にした場合、本サービスの一部の機能を利用できなくなることがあります。
  4. 運営者がアクセス解析サービスを利用する場合、当該サービスの提供者がCookie等を用いてアクセス情報を取得することがあります。

第9条（情報の安全管理）

運営者は、個人情報への不正アクセス、漏えい、滅失、毀損、改ざんその他の問題を防止するため、次のような合理的な安全管理措置を講じるよう努めます。

  1. 個人情報にアクセスできる者の制限
  2. アカウントおよび管理者権限の適切な管理
  3. パスワードその他の認証情報の適切な保護
  4. 通信の暗号化その他の技術的な安全対策
  5. 不正アクセスおよび不審な操作の確認
  6. 使用するシステムおよび外部サービスの適切な選定
  7. 個人情報が不要となった場合の削除または匿名化
  8. 漏えい等が発生した場合の確認および対応体制の整備

ただし、インターネット上における情報の安全性を完全に保証することはできません。

第10条（情報の保存期間）

  1. 運営者は、個人情報を、その利用目的を達成するために必要な期間保存します。
  2. 利用者がアカウントを削除した場合、運営者は、法令上またはサービス運営上保存する必要がある情報を除き、合理的な期間内に個人情報を削除または匿名化します。
  3. 利用規約違反への対応、不正行為の防止、紛争への対応、法的義務の履行その他正当な理由がある場合、アカウントの削除後も必要な情報を一定期間保存することがあります。
  4. バックアップデータに含まれる情報については、システム上可能な時点で順次削除される場合があります。

第11条（アカウントの削除）

  1. 利用者は、運営者が定める方法により、アカウントの削除を申し込むことができます。
  2. アカウントを削除した場合、利用者は、本サービス上の登録情報、出品情報、メッセージその他の情報を利用できなくなることがあります。
  3. 取引中の商品がある場合、利用者は、原則として当該取引を完了またはキャンセルした後にアカウントの削除を申し込むものとします。
  4. アカウント削除後も、他の利用者とのメッセージ、取引記録、通報記録その他不正防止または紛争対応のために必要な情報が保存される場合があります。

第12条（開示、訂正、削除等の請求）

利用者は、法令に基づき、運営者が保有する自己の個人情報について、利用目的の通知、個人情報または第三者提供記録の開示、内容の訂正・追加・削除、利用の停止・消去、第三者への提供の停止を請求できます。

請求を希望する場合、利用者は、第17条に定める問い合わせ先まで連絡するものとします。運営者は、請求者が本人であることを確認した上で、法令に従い合理的な期間内に対応します。ただし、法令により請求に応じる義務を負わない場合は、その全部または一部に応じないことがあります。

第13条（利用者の責任）

  1. 利用者は、他の利用者から取得した氏名、連絡先、待ち合わせ情報その他の個人情報を、当該取引のために必要な範囲でのみ利用するものとします。
  2. 利用者は、取引相手の個人情報を、本人の同意なく第三者に提供、公開または拡散してはなりません。
  3. 利用者は、取引終了後、保存する必要のない取引相手の個人情報を適切に削除するものとします。
  4. 利用者間で交換された個人情報を利用者が不適切に使用したことによって生じた問題については、当該利用者が責任を負うものとします。
  5. 運営者は、運営者の責めに帰すべき事由がある場合を除き、利用者が本サービス外で個人情報を交換したことによって生じた問題について責任を負いません。

第14条（未成年者の情報）

未成年者が本サービスを利用する場合は、必要に応じて親権者その他の法定代理人の同意を得た上で、個人情報を提供するものとします。

第15条（大阪大学との関係）

  1. 本サービスは、大阪大学が公式に運営、提供または保証するサービスではありません。
  2. 運営者は、利用者から取得した情報を、大阪大学に当然に提供するものではありません。
  3. ただし、重大な規則違反、犯罪行為、人の生命または身体に対する危険その他の事情があり、法令上認められる場合には、必要な範囲で大阪大学、警察その他の関係機関に情報を提供することがあります。

第16条（本ポリシーの変更）

  1. 運営者は、法令の改正、本サービスの内容変更その他必要がある場合、本ポリシーを変更することがあります。
  2. 重要な変更を行う場合、運営者は、変更後の内容および適用開始日を、本サービス上への掲載その他適切な方法によって利用者に通知します。
  3. 変更後の本ポリシーは、本サービス上に掲載された日または運営者が別途定めた適用開始日から効力を生じます。

第17条（お問い合わせ窓口）

本ポリシーおよび個人情報の取扱いに関する問い合わせ、相談、苦情、開示等の請求は、次の窓口で受け付けます。

運営者名：pbl5班
サービス名：OU Textbook
問い合わせフォーム：ヘルプ／お問い合わせ
受付方法：原則として本サービス内の問い合わせフォーム
